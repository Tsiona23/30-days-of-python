## features of python 
  * easy to learn
   it uses simple english line comands.
  * high-level language
    you don't need to manage memory or hardware details.
     - no pointers like c++.
     - no complex syntax.
  * interprated language
    python runs line by line using an interprater.
    you don't need to compile first.
  * object-oriented
   python supports classes and objects.
  * portable
  python programs run on : 
        - windows
        - linux
        - mac  
  * open source
   python is free to use
  * large library support
    python has many built in libraries like:
         -math operating
         -file handling
         -networking
         -web developmment
         -AI and machine learning
